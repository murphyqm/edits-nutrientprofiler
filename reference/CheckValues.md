# Check parameter values function

A function that takes an input dataframe and counts unique occurrences
for each parameter and lists all included values. This allows you to
check that no values have misspelled or invalid entries.

## Usage

``` r
CheckValues(data_frame)
```

## Arguments

- data_frame:

  a data.frame object with product data

## Value

results a data.frame object with each named parameter and the unique
occurrences of values
