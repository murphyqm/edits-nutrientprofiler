# Save report function

This function saves a transposed version of the report generated by
[`ReqParamCheck()`](https://leeds-cdrc.github.io/nutrientprofiler/reference/ReqParamCheck.md)

## Usage

``` r
SaveReport(data_frame, file_path)
```

## Arguments

- data_frame:

  a data.frame object

- file_path:

  a chr object containing relative filepath from the current working
  directory, to save csv

## Value

saves csv file
