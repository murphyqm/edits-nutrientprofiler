# C score function

C score is defined as the scores for fruit/veg/nut percentage, protein
and fibre summed together

## Usage

``` r
C_scorer(fvn_score, protein_score, fibre_score)
```

## Arguments

- fvn_score, :

  numeric value for fruit/vegetables/nuts percentage score

- protein_score, :

  numeric value for protein score

- fibre_score, :

  numeric value for fibre score

## Value

a numeric value of the C score
