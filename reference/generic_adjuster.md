# Function for generically adjusting nutritional measurement by total weight

Adjustment is required for calculating scores based on nutritional
measurements.

## Usage

``` r
generic_adjuster(value, adjusted_weight)
```

## Arguments

- value:

  a numeric value corresponding to a nutritional measurement in a
  food/drink

- adjusted_weight:

  a numeric value corresponding to the total weight of the food/drink
  after specific gravity adjustment

## Value

a numeric value of adjusted nutritional data
